# Hyperspace & Hyperdrive

Hyperspace is EPIC's primary desktop application -- the main interface that clinicians, staff, and administrators use daily. Hyperdrive is the newer, web-based version that is gradually replacing Hyperspace.

## Hyperspace (Desktop Client)

- Traditional installed Windows application
- Connects to the EPIC server over your hospital's network
- Has been the standard interface since EPIC's early days
- Uses a tab-based layout with customizable toolbars

## Hyperdrive (Web Client)

- Browser-based version of the EPIC interface
- Does not require a local installation
- Enables access from more device types
- Becoming the default at many organizations

## Key Interface Elements

### The Toolbar
The top toolbar provides quick access to common functions: patient lookup, schedule, InBasket, and more. It is customizable per role.

### Activity Tabs
EPIC uses an activity-based navigation model. Each "activity" (Chart Review, Order Entry, Notes, etc.) opens in a tab. You can have multiple activities open simultaneously.

### The Patient Header
When you have a patient chart open, the header bar shows key demographics: name, MRN, date of birth, allergies, and code status. This header persists across all activities for that patient.

### The Navigator/Sidebar
A sidebar that provides quick links to different sections within the current patient's chart. The contents change based on context and your role.
